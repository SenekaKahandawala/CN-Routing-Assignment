## Cisco Packet Tracer Networking Assignment

# Overview
This project demonstrates fundamental networking concepts using Cisco Packet Tracer. It includes router configuration, static routing, and dynamic routing using RIP and EIGRP protocols.

# Tasks Covered

## Task 1: Basic Configuration
- Router setup and initial configuration
- IP addressing for interfaces
- Basic connectivity setup

## Task 2: Static Routing
- Configuration of static routes between multiple networks
- Manual route definition between routers
- Verification using ping and routing table commands

## Task 3: Dynamic Routing
- Implementation of RIP (Routing Information Protocol)
- Implementation of EIGRP (Enhanced Interior Gateway Routing Protocol)
- Automatic route exchange between routers
- Connectivity verification using ping tests and routing table analysis

# Technologies Used
- Cisco Packet Tracer
- RIP (RIPv2)
- EIGRP
- IPv4 Networking

# Key Learning Outcomes
- Understanding of router configuration
- Difference between static and dynamic routing
- Hands-on experience with routing protocols
- Network troubleshooting and verification
